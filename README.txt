COLLABORATIVE PSYCHIATRIC EPIDEMIOLOGY SURVEYS (CPES)

The aim of this report is to explore an open source database containing information about a sample of American citizen's physical and mental health, as well as demographic information.
First, the data will be explored using clustering, and then hypothesis testing.

Plan
1. Pre-process data, decide on which variables to include
2. Exploratory analysis: Conduct correspondance analysis (equivalent to PCA)
3. Hypothesis testing
